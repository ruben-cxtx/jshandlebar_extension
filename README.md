# js-to-handlebars-parser

VS Code extension that transforms JavaScript into Handlebars templates for SendGrid.

## Features

- Convert selected JavaScript into Handlebars syntax.
- Quick command for transforming the current file.

## Requirements

- VS Code 1.90 or newer.

## Extension Settings

This extension currently does not contribute any settings.

## Known Issues

- Nested template literals may require manual cleanup.
